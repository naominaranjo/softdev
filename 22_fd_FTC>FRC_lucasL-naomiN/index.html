Welcome to Foundation!

To get foundation working, simply link to the following CSS link https://cdn.jsdelivr.net/npm/foundation-sites@6.7.4/dist/css/foundation.min.css
You may also want to link the core foundation JavaScript library at https://cdn.jsdelivr.net/npm/foundation-sites@6.7.4/dist/js/foundation.min.js
Links to any needed utilities can be found at https://cdn.jsdelivr.net/npm/foundation-sites@6.7.4/dist/js/plugins/
Documentation on different foundation components can be found at https://get.foundation/sites/docs/
For JavaScript, make sure that you include a script tag at the end of the page with $(document).foundation();
Inspect and take a look for the components for a look at some examples
